# LabFrame_ExpressDetect
> com.xrlab/labframe_expressiondetect

## Summary
- Working on HTC Vive Focus3 VR device
- To get and deal with data about eye and mouth expressions
- Warning: There's no eye tracking in this project!!!

- HTC Vive Focus3 VR頭盔的眼部表情與嘴部表情追蹤，沒有眼動ㄡ

## Vive Wave Setup

This package keeps Wave dependencies optional. If you are targeting Vive Focus devices,
install Wave support from Unity > Window > Package Manager:

- Install `Vive Wave XR Plugin`.
- Install `Vive Wave XR Plugin - Essence`.
- Install `Vive Wave XR Plugin - Native`.
